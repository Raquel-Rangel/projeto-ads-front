# Projeto ADS: Página de Contato Interativa

Este projeto consiste na criação de uma página web para um formulário de contato, com foco em **validação em tempo real** e **design responsivo**, utilizando HTML, CSS e JavaScript. O objetivo foi desenvolver uma interface de usuário amigável e funcional, garantindo a integridade das informações submetidas.

## Funcionalidades Implementadas:

1. **Cabeçalho e Rodapé Estilizados:**

    * Design profissional com o nome do serviço/empresa ("TechSolutions - Fale Conosco").

    * Estilização que acompanha o tema visual da página.

2. **Formulário de Contato Completo:**

    * **Campos:**

        * Nome (obrigatório, mínimo 3 caracteres)

        * E-mail (obrigatório, formato válido)

        * Telefone (com máscara de entrada)

        * Assunto (select com opções: Suporte, Vendas, Reclamação, Outros)

        * Mensagem (textarea, obrigatória, mínimo 10 caracteres)

        * Checkbox para aceitar termos de privacidade (obrigatório)

3. **Validação em Tempo Real com JavaScript:**

    * Feedback visual imediato para campos obrigatórios e formatos incorretos.

    * Verificação dinâmica de e-mail e telefone.

    * Confirmação visual (cores, ícones) para campos preenchidos corretamente ou com erros.

    * Mensagens de erro/sucesso dinâmicas e claras.

4. **Estilização Responsiva com CSS:**

    * Utilização de **Flexbox/Grid** para garantir que o layout se adapte perfeitamente a diferentes tamanhos de tela (desktops, tablets, smartphones).

    * Design visualmente agradável e organizado.

5. **Interatividade e Submissão:**

    * Botão de envio que, ao ser clicado, realiza a validação final.

    * Exibição de uma mensagem de confirmação clara (Ex: "Dados enviados com sucesso!") ao final do processo.

## Tecnologias Utilizadas:

* **HTML5:** Para a estrutura semântica da página.

* **CSS3:** Para estilização e responsividade (Flexbox/Grid).

* **JavaScript:** Para validação em tempo real, manipulação do DOM e interatividade.

## Pontos de Destaque e Aprendizado:

* Aprofundamento em **validação de formulários com JavaScript**, incluindo validações de formato e obrigatoriedade.

* Prática com **design responsivo** e técnicas de layout (Flexbox/Grid).

* Melhora na **experiência do usuário (UX)** através de feedback visual dinâmico e máscaras de entrada.

* Compreensão da importância de cada etapa: estrutura (HTML), apresentação (CSS) e comportamento (JavaScript).
